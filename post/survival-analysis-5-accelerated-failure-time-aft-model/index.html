Survival Analysis 5: Accelerated Failure-time (AFT) model

2023-11-22 · 17 min read

If we want to find the relationship between hazard function and other variables, we can use the Cox proportional hazards (PH) model. However, what if we want to model the survival time itself? In this case we can use Accelerated Failure-time (AFT) models.

Introduction

The AFT models is used to model the effect of the covariates on the survival time. Its rudimentary mathmatics form is as below:

T=T0exp(zβ)T = T_{0}exp(z \beta) T=T0​exp(zβ)

or equivalently

lnT=lnT0+zβlnT = lnT_{0} + z \beta lnT=lnT0​+zβ

  • T is a vector of survival times.
  • z is a vector of covariates.
  • β\betaβ is a vector of unknown regression parameters.
  • T0T_{0}T0​ is the survival time distribution at baseline, when covariates z are zero. In AFT model, the lnT0lnT_{0}lnT0​ is treated as random variable.

If we comepare with the Cox PH model

λ(t∣z)=λ0(t)exp(zX)\lambda (t|z) = \lambda_{0}(t) exp(z X) λ(t∣z)=λ0​(t)exp(zX)

we can find that the interpretation of β\betaβ is different. For PH model, β\betaβ > 0 implies an increased hazed, shorter survial time; whereas it is opposite for AFT models.

Model tuning

If we introduce intercept term α\alphaα and a scale parameter δ\deltaδ to the model:

lnT=α+zβ+δlnT0lnT = \alpha + z\beta + \delta lnT_{0} lnT=α+zβ+δlnT0​

The original failture time is in the form of

T=exp(α+zβ+δlnT0)=T0δexp(α)exp(zβ)T = exp(\alpha + z\beta + \delta lnT_{0}) = T_{0}^{\delta} exp(\alpha) exp (z\beta) T=exp(α+zβ+δlnT0​)=T0δ​exp(α)exp(zβ)

The effect of the intercept and scale is to power the survival time. In other words, the effect of covariates in AFT is to affect the scale but not the location of a baseline survival time distribution.

Such model can also be rewritten more akin to linear regression model in the form of

lnT=zβ+δϵlnT = z\beta + \delta \epsilon lnT=zβ+δϵ

  • where the intercept is incorporated in the coefficient vector β\betaβ.

Here comes to the key point of all "fuss" of doing all the transformation above.

We can further assume different distribution of the random error term by defining density function of the error term ϵ\epsilonϵ. It is also called extended generalized-gamma (EGG) model.

  • When q = 0

f(q,ϵ)=12πexp(−ϵ22)f(q, \epsilon) = \frac{1}{\sqrt{2 \pi}} exp (-\frac{\epsilon^{2}}{2}) f(q,ϵ)=2π​1​exp(−2ϵ2​)

  • When q ≠=0\neq = 0​==0

∣q∣Γ(q−2)(q−2)q−2exp[q−2(qϵ−exp(qϵ))]\frac{|q|}{\Gamma(q^{-2})} (q^{-2})^{q^{-2}} exp[q^{-2}(q \epsilon - exp(q \epsilon))] Γ(q−2)∣q∣​(q−2)q−2exp[q−2(qϵ−exp(qϵ))]

The advantage of setting density function in such way is that when we set different values for shape parameter q, we can get different distribution for the error term.

  • When q = 0, T has a log-normal distribution.
  • When q = 1, T has a Weibull distribution.
  • When q = 1 and ϵ=1\epsilon = 1ϵ=1, T has a exponential distribution which is a special case of Weibull distribution.
  • When q = -1 and ϵ=1\epsilon = 1ϵ=1, T has a reciprocal-Weibull distribution.
  • When ϵ=1\epsilon = 1ϵ=1, T has a ordinary gamma distribution.

These 5 models above are special cases of the EGG model. Another model of interest, though not a special case of the EGG model, is the log-logistic model.

In R, we can In R we can do Weibull, Exponential, Gaussian, Logistic, Lognormal. However, reciprocal-Weibull distribution and EGG are not available in R unfortunately.

Understanding the setting above can also help us interpreting the R results properly.

Practice in R

In R we can use the survreg() function in survival package to estimate the AFT model.

Model estimation

In the summary() output you can see output of "Newton-Raphson". "Newton-Raphson" is the method used for maximizing the likelihood. More specifically:

  • Consider survival times of n individuals to be t1t_{1}t1​, t2t_{2}t2​..., tnt_{n}tn​ and p covariates z1z_{1}z1​, z2z_{2}z2​, ...., zpz_{p}zp​.
  • Let did_{i}di​ = 0 if tit_{i}ti​ is consoring time and did_{i}di​ = 1 if tit_{i}ti​ is time to the event.
  • The log-likelihood function ln(lnt;zβ,δ,q)ln(lnt;z\beta,\delta, q)ln(lnt;zβ,δ,q) assumes a noninformative consoring mechanism thus it is proportional to

∑indi[lnf(q,ϵi)−lnδ]+∑i=1n(1−di)lnS(q,ϵi)\sum_{i}^{n} d_{i} [lnf(q, \epsilon_{i}) - ln \delta] + \sum_{i = 1}^{n} (1 - d_{i}) lnS(q, \epsilon_{i}) i∑n​di​[lnf(q,ϵi​)−lnδ]+i=1∑n​(1−di​)lnS(q,ϵi​)

where ϵi=lnTi−ziβδ\epsilon_{i} = \frac{lnT_{i} - z_{i}\beta}{\delta}ϵi​=δlnTi​−zi​β​ and S(ϵ,q)S(\epsilon, q)S(ϵ,q) is the corresponding survivor function. And "Newton-Raphson" is the method used for maximizing the likelihood for the equation above. One can see Allison (2010) about carrying out the algorithm in SAS.
